

This attached shell script will take a Fortify SCA build id and a source directory and then check to see which files were not translated.

Example Process:
=================

The example usage uses the Samples directory from Fortify and performs a translation without any build command.  Located at <Fortify SCA Install DIR>/Samples

sourceanalyzer -b samples -verbose Samples

verifyTranslation.sh samples Samples
